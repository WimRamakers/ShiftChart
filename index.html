Shiftchart

A simple JavaScript webapp to generate nice, and printable, rpm vs speed curves for all gears of your car/bike/lawnmower on steroids.

View project onGitHub

What & why

This project started when I was learning JavaScript/jQuery, and wanted a real world challenge. Since my tool to print such images was Microsoft Excel at that time, I decided to expand the functionality and make it platform independent as a web app. I'm now going open source because other people might have good ideas for new features and want to help to create the ultimate tool. I must warn you guys, the code is not very clean since a lot of work was done at the time i was just learning Javascript...

Implemented features

  • sliders to see what influence changes have just by dragging a slider
  • torque curve based optimal acceleration shift point calculation for those who think that the rpm limiter is always the best shift point
  • multiple setups can be compared
  • values on image when only one setup
  • saving of setups and views (=some setups together) as private or public when an account is generated
  • print stylesheet
